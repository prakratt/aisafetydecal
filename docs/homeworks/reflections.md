## Weekly Reflections

Reflections are due **before each Wednesday class**. Each reflection should be roughly **300–500 words** and cover the assigned before-class reading for that week.

### Standard Reflection Prompts

For every week, write a short response addressing the following:

1. **Summary.** In 2–3 sentences, what was the central claim of the reading?
2. **Strongest point.** Which argument did you find most compelling, and why?
3. **Weakest point / disagreement.** Where did the author's reasoning feel weakest? What would you push back on?
4. **Connections.** How does this reading connect to a previous week's material, another paper you've read, or a current event?
5. **Open question.** What's one question you'd want to discuss in class?
